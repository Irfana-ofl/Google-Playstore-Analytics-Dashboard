## 📊 Google Play Store Analytics Dashboard

An interactive data analytics dashboard developed as part of a Data Analytics Internship, focused on analyzing Google Play Store datasets to uncover insights about app performance and user behavior.  <br>
The project demonstrates structured data processing, advanced filtering, and time-based visualization techniques using Python and Plotly.

✔ Real-world dataset analysis <br>
✔ Interactive and dynamic visualizations <br>
✔ Focus on analytical thinking and data storytelling

---

## 📌 Internship Overview

**Internship Title:** Learn to Build Real-Time Google Play Store Data Analytics using Python <br>
**Duration:** 1 Month (21 Nov 2025 – 25 Dec 2025) <br>
**Completion Status:** 100% Completed <br>
**Tasks Completed:** 6 / 6 <br>

This internship focused on building a real-time analytics dashboard by applying data cleaning, visualization, and advanced filtering techniques on Google Play Store datasets.

## 📁 Datasets Used

• **Play Store Data.csv** <br>
→ Contains app details such as category, rating, installs, size, reviews, and last updated

• **User Reviews.csv** <br>
→ Includes sentiment data such as polarity and subjectivity

## ⚙️ Project Workflow

1. Data Collection  
   → Imported Google Play Store datasets  

2. Data Cleaning & Preprocessing  
   → Handled missing values, removed inconsistencies, formatted columns  

3. Data Transformation  
   → Applied filters based on ratings, installs, size, and categories  

4. Sentiment Analysis  
   → Analyzed user reviews using polarity and subjectivity  

5. Data Visualization  
   → Built interactive charts using Plotly  

6. Dashboard Development  
   → Integrated visualizations into a structured dashboard using HTML & CSS  

7. Deployment  
   → Hosted the dashboard using GitHub Pages  

---

## 📈 Key Insights

• App categories with high installs do not always have the highest ratings  
• Free apps dominate in total installs, while paid apps contribute more to revenue  
• Positive sentiment in reviews generally aligns with higher app ratings  
• Certain categories show noticeable growth trends over time  
• Filtering reveals that app size and updates can influence user engagement
